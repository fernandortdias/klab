CONVERT PROGRAM VERSION 2006
This program helps to convert ArcInfo ASCII grid files into a tabular format for analysis in spreadsheets or statistical packages such as SPSS.

Files needed for proper functioning:

names.txt:
The lines each specify the full name of files that need to be converted. This file can be edited trhough the interface or as text file. After the last file name an 'enter' is not necessary.

help.txt:
This file

Output:
stat.txt combines all files in a tabular format

This program can also be run in Batch mode by calling the executable followed by the settings for the program. These should include in the following order: balanced (1) or unbalanced (0); exclusion of nodata (1) or inclusion (0); the mimimal distance; the sample size (%).
