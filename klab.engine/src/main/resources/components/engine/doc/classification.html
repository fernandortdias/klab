Classification

A classification maps an input value to an output value.

#foreach( $entry in $classification ) #end
Input Output
${entry.Second.HtmlSourceCode} ${entry.First.HtmlDeclaration}

Each input is matched from the top: the value corresponding to the first match is returned. If the input is a concept, it will be matched using the reasoner, so that more generic concepts will match a more specific one. A star (*) matches any input value but not no-data; a pound (#) will match anything including no-data.
