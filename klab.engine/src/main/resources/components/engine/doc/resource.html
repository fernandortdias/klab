This processing step retrieves the contents of a data or model resource from the semantic web. Resources can be data files, data services (using protocols such as OGC or OpenDAP), or may interface to more complex computations or running simulations.

Resources are identified by a unique Uniform Resource Name (URN) used together with the scale of observation to retrieve data or trigger computation. Metadata and provenance records associated with this resource are shown below.
